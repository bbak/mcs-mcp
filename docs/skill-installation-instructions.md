# Installing Skills

After downloading and extracting the MCP Server archive, follow these steps to install the skills in Claude.

**Note**: This may be different for other Agents/Assistants that already support skills.

- Claude Desktop stores skills in Anthropics infrastructure. There's no local directory. Use the Claude Desktop UI to upload/install SKILLs.
- Claude Code does have a local `skills` folder.

## Prerequisites

- Node.js installed, which is required to render JSX/React components from skills.
